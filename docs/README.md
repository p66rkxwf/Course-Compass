# Course Master 專案說明

智慧選課輔助系統，提供完整的課程資料分析與選課輔助功能。

## 主要功能

### 資料爬取
- 自動爬取學校課程資料
- 支援多學期批量爬取
- 自動處理 ASP.NET 表單
- 解析課程表格與教學大綱連結

### 資料處理
- **課程名稱分割**：自動分離中英文課程名稱
- **教師姓名解析**：使用字典與演算法智能分割
- **時間地點解析**：處理複雜的上課時間格式
- **科系映射**：從開課班別提取學院、科系、年級、班級資訊
- **資料驗證**：確保資料品質與完整性

### API 服務
- 課程搜尋與過濾
- 統計資料獲取
- 課程詳情查詢
- 智慧推薦系統
- 歷年資料查詢
- 支援 CORS 與靜態檔案服務

### Web 介面
- **完整課表系統**：12 節次課表，支援多節次課程，視覺化顯示
- **一鍵導入功能**：選擇系所與班級後，一鍵導入該班級所有必選修課程
- **智慧推薦系統**：根據空堂時段、目標學分數、課程類別智能推薦
- **歷年資料查詢**：查詢課程與教師的歷史開課記錄，包括選課人數趨勢
- **缺額追蹤功能**：追蹤爆滿課程，自動監控人數變化，有人退選立即通知
- **本地儲存**：課表資料自動儲存到瀏覽器本地儲存

- **科技學院**：電機與機械科技學系、智慧車輛工程學系等

## 使用說明

### 啟動服務器

```bash
python main.py api
```

### 訪問網站

打開瀏覽器訪問：http://localhost:8000

### 主要操作流程

1. **選擇班級並導入課程**
   - 選擇學年度與學期
   - 選擇系所
   - 選擇班級
   - 點擊「一鍵導入」按鈕

2. **查看與調整課表**
   - 在「我的課表」頁面查看已選課程
   - 點擊課程卡片查看詳情
   - 可以手動移除不需要的課程

3. **使用智慧推薦**
   - 切換到「智慧推薦」頁面
   - 設置目標學分數
   - 選擇課程類別
   - 點擊「開始推薦」
   - 從推薦結果中選擇課程加入課表

4. **查詢歷年資料**
   - 切換到「歷年大數據」頁面
   - 輸入課程名稱或教師姓名
   - 查看歷年開課記錄

5. **追蹤課程缺額**
   - 在推薦結果中點擊「追蹤」按鈕
   - 切換到「缺額追蹤」頁面查看追蹤列表
   - 系統會自動監控，有人退選時會通知

## 技術架構

### 前端技術
- **Bootstrap 5.3.2** - 響應式 UI 框架
- **Bootstrap Icons** - 圖標庫
- **Font Awesome 6.4.0** - 額外圖標
- **原生 JavaScript** - 無依賴，輕量高效

### 後端技術
- **FastAPI** - 現代化 Python Web 框架
- **Pandas** - 資料處理
- **JSON 檔案儲存** - 追蹤資料持久化

### API 端點

#### 課程相關
- `GET /api/courses/all` - 獲取所有課程
- `GET /api/courses/search` - 搜尋課程
- `GET /api/courses/by-class` - 根據班級獲取課程
- `POST /api/courses/recommend` - 推薦課程
- `GET /api/courses/history` - 歷年資料查詢
- `GET /api/courses/stats` - 統計資訊
- `GET /api/courses/{course_id}` - 課程詳情

#### 系所相關
- `GET /api/departments` - 獲取系所列表
- `GET /api/departments/colleges` - 獲取學院列表
- `GET /api/departments/stats` - 獲取部門統計資訊

## 注意事項

1. **資料載入**：首次使用需要確保已處理課程資料（執行 `python main.py process`）
2. **瀏覽器相容性**：建議使用現代瀏覽器（Chrome、Firefox、Edge 最新版本）
3. **本地儲存**：課表資料儲存在瀏覽器本地儲存中，清除瀏覽器資料會遺失
